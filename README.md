# Image Gallery App

A responsive React + TypeScript application that allows users to:
- Enter their name
- Browse images from API
- Save selected images
- View saved images

## Features
- React Router navigation
- Context API integration
- Pagination (max 5 pages)
- Responsive design (mobile + desktop)

## Tech Stack
- React
- TypeScript
- Vite
- Context API

## Run Locally
npm install
npm run dev
